# Wirefall Sannabi-like Roguelike

총격과 와이어 기동을 중심으로 한 사이드 스크롤 로그라이크 웹 게임 프로토타입입니다.

## Play

- 이동: `A/D` 또는 방향키
- 점프: `Space`, `W`, 또는 위 방향키
- 사격: 마우스 좌클릭 또는 `J`
- 와이어: 마우스 우클릭 또는 `E`
- 강화 선택: `1-3`
- 재시작: `R`

## Local Run

```bash
python -m http.server 4173
```

브라우저에서 `http://localhost:4173/`를 열면 됩니다.
